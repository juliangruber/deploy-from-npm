
# deploy-from-npm

  WIP Continuous deployment tool tailing npm.

## Usage

```bash
$ deploy-from-npm

Usage:

  $ deploy-from-npm DIR RELOAD

Example:

  $ deploy-from-npm /srv/my-service "kill $server-pid"

```

## Installation

```bash
$ npm install -g deploy-from-npm
```
